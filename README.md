## How to Run
### To run dev build (local):
```
git clone https://github.com/zumbiepig/MineXLauncher
npm install
npm start
```

#### go to https://localhost:3000
==============================================================
### To run stable (local):
Download zip from releases
unzip it in the folder you want
```
npm install
npm start
```
#### go to localhost:3000
=============================================================
### Host on pages:
fork repo
set up pages
### Host on Vercel
fork repo
go to https://vercel.com
set up from there
